# PV-Student Council Platform

Platform for Student Council elections and information.

## Overview

This is a web-based platform for the student council, featuring:

- **Timeline**: Updates on policies and news.
- **Candidates**: Information about parties and members.
- **Vote**: Election information.

## How to use

This is a static web page. You can simply open `index.html` in your web browser.

## Technologies

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
